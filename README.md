# Amazon UI Automation
This project automates the UI interactions on the Amazon website using Selenium and Java. It includes test cases for product search, adding items to the cart, and checkout validation.

## Tech Stack
- **Programming Language:** Java  
- **Automation Framework:** Selenium WebDriver
- **Test Data:** Excel  
- **Testing Framework:** TestNG  
- **Build Tool:** Maven  
- **Reporting:** Extent Reports
